README for GitHub Repository CJZ-Western-toad-model
Nov 20, 2025

The files in this repository produce Figures 4-7 in the paper:

Marguerite H. Mahr, Noah D. Marshall, Jessa Marley, Sarah K. Wyse, Wayne P. McCrory, Rebecca C. Tyson, Modelling road mortality risks to persistence to a 
Western Toad (Anaxyrus boreas) population in British Columbia. Canadian Journal of Zoology (accepted: Oct 2025) https://arxiv.org/html/2510.00340v1

The files were written in Matlab R2024a (24.1.0.2537033), 64-bit (glnxa64), Feb 21, 2024.

The code was written by Noah Marshall & Rebecca Tyson.
